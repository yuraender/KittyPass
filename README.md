# KittyPass

**Менеджер паролей** для Дани личного использования и с любви к котикам.

## Возможности

- Локальное хранение паролей в SQLite базе данных на вашем компьютере
- Импорт и экспорт паролей в формате JSON
- Красивый кастомный интерфейс с настраиваемой темой

## Технологии

- **Frontend**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Electron
- **База данных**: SQLite

## Установка и запуск

### Предварительные требования

- Node.js (версия 22 или выше)
- npm или pnpm

### Сборка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка с установщиком
npm run build
